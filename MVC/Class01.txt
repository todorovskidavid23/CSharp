kako odi procesot na MVC

1.User interacts with the VIEW
VIEW sends request to CONTROLLER
CONTROLLER processes the request
CONTROLLER communicates with MODEL
MODEL retrieves data
MODEL returns data to CONTROLLER
CONTROLLER updates the VIEW
User sees the result

1. vo view imame odredeni kopcinja kako user od view samo gledam i iklikam da se najavam na nekoja web starnica
2. vie go konektira controoelrot e HTTP baranje GET
3. controlerot odlucuva sto ponataka gleda toj sto kliknal ima moznost taa funkcija da se izvrsi
4.ako smee si prdoulzuva si se dvizi vo samiot controlet, pusta baranje i go nosi do modelot mu prakja do moedlot
5.modelot go obrabotuva baranjeto gi sobira podatocite i pocnuva kon vrakjanje na podatocite do userot
